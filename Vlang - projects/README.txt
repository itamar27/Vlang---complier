1. A file with a vlang phrases is added to this project, its name is:
	"vlang.in"

2. Inorder to to translate the langauge please run the next command:
	.\vlang.exe .\input.in .\vlang.c

   (You can also run the vlang.exe command alone but it will out put to the screen);

3.After you will run this command you will see a new file in your directory 
  with the name "vlang.c"

4. inorder to see the result please compile the "vlang.c" file ->
	gcc  vlang.c -o out.exe

5. now you can run your new program and test the result with:
	.\out.exe
